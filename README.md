# She Can Foundation

A responsive NGO website built using HTML, CSS and JavaScript.

## Features

- Responsive Design
- Dark Mode
- Volunteer Registration Form
- Form Validation
- Animated Counters
- Success Stories Section
- Programs Section
- Contact Section
- Admin Dashboard
- Volunteer Analytics
- CSV Export
- Search Functionality
- Back To Top Button

## Tech Stack

- HTML5
- CSS3
- JavaScript
- LocalStorage
